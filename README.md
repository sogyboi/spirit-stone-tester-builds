- # Spirit Stones Remake 0.1.6 Alpha
- ## Faster menus, cleaner boot, and safer loading
- Loading and menu have been dramatically improved from over 14 seconds down to almost 2. this is from splash screen all the way to main menu.
- Large card libraries should be much less should now load fully and not lag on page changes.
- ## Battle Mode and PvP groundwork
- Battle Mode and PvP have had a lot of system work.
- The game now has more of the pieces needed for "player-vs-player" style battles, live battle flow, rival turns, and simulated battle checks. This still needs testing, but the foundation is much stronger than it was in 0.1.2.
- Combat board rules were also tightened up. no more connecting bombs, swords and potions all at the same time :O
- ## Combat and reward polish
- Reward screens should feel cleaner, there was a bug/ visual issue that one of the chest drops was a card drop when it was actually just saying "you got a card pull for completing this stage" it was confusing and not even a reward its now gone.
- Random Box and Super Random Box behavior was improved too. They can now open in larger batches more like the chest reward flow, instead of feeling like you have to tap through everything one at a time.
- There is also new account level-up popup plumbing. When battle EXP causes the account level to go up, the popup is handled back on the main menu instead of inside the battle. For now it shows the level change and EXP progress only. No extra level-up rewards are being invented yet until there is a real reward table for it.
